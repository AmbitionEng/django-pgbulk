Documentation
=============

`View the django-pgbulk docs here
<https://django-pgbulk.readthedocs.io/>`_.

Installation
============

Install django-pgbulk with::

    pip3 install django-pgbulk

After this, add ``pgbulk`` to the ``INSTALLED_APPS``
setting of your Django project.
